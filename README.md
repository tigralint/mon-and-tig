# ✦ Lumea — AI-платформа для осмысленного обучения

**Lumea** — это интеллектуальная платформа, которая превращает хаос учебных материалов в структурированную, живую систему знаний. Вместо того чтобы просто хранить конспекты, Lumea _понимает_ их: извлекает смысл, находит связи между дисциплинами и помогает запоминать по научно обоснованным методикам.

> Lumea — от латинского _lumen_ (свет). Платформа буквально подсвечивает знания, которые скрыты в ваших документах.

---

## Проблема

Студенты тонут в материалах: десятки PDF-лекций, конспектов, учебников — всё это лежит мёртвым грузом в папках. Перечитывать неэффективно. Ctrl+F ищет слова, а не смысл. Связи между предметами теряются. Повторение хаотично.

## Решение

Lumea — это **персональный академический AI**, работающий прямо в браузере. Вы загружаете документы, а платформа:

1. **Понимает содержание** — парсит PDF/DOCX, OCR-распознаёт сканы, разбивает на семантические фрагменты, генерирует векторные эмбеддинги
2. **Отвечает на вопросы по вашим материалам** — RAG-чат, где источником истины являются _только ваши_ документы, а не галлюцинации модели
3. **Создаёт флеш-карточки** — автоматически из текста или фото конспекта, с интервальным повторением по алгоритму SM-2
4. **Находит неожиданные связи** — Serendipity Engine ищет пересечения между разными документами и формулирует междисциплинарные инсайты
5. **Визуализирует ваши знания** — 3D-граф «Созвездие знаний», где каждый документ — звезда, а связи между ними — реальная семантическая близость

## Архитектура

### Privacy-first: AI работает на устройстве

Ключевое архитектурное решение — **локальные эмбеддинги**. Для семантического поиска и индексации используется модель [EmbeddingGemma 300M](https://huggingface.co/onnx-community/embeddinggemma-300m-ONNX) от Google DeepMind, которая работает прямо в браузере через ONNX/WebAssembly. Это значит:

- 📴 Поиск и индексация работают **полностью офлайн** (после первой загрузки модели)
- 🔒 Текст документов **не покидает устройство** при индексации
- 🌍 Поддержка 100+ языков, включая русский, английский, немецкий и французский

Для генеративных задач (конспектирование, RAG-ответы, создание карточек) используется **Gemma 4 31B** через Google Generative AI API, с возможностью проксирования через Cloudflare Worker для защиты API-ключа.

### Стек

| Слой | Технология |
|------|-----------|
| UI | React 19, React Router, Lucide Icons |
| Сборка | Vite 8 |

| State | Zustand |

## Ключевые модули

### 📄 Умная обработка документов
- **PDF** — извлечение текста через pdfjs, автоматическое определение сканированных страниц
- **OCR** — Tesseract.js (WASM): 4 языка (eng/rus/deu/fra), предобработка (grayscale + контраст + бинаризация), адаптивный масштаб рендера, невидимый текстовый слой поверх сканов для нативного выделения текста
- **DOCX** — парсинг через Mammoth.js в Web Worker
- Прогресс обработки в реальном времени: `📄 Читаем PDF → 🔤 OCR: стр.3 (2/5) → ✅ Готово`

### 🔍 RAG-чат — «Спроси свои документы»
Задайте вопрос на естественном языке, и Lumea:
- Найдёт релевантные фрагменты через семантический поиск (cosine similarity по локальным эмбеддингам)
- Подтянет контекст из «Умной памяти» — фактов о вашем стиле обучения, которые система накапливает со временем
- Сгенерирует ответ через LLM, строго на основе ваших документов, с указанием источников

### 🧠 Флеш-карточки с интервальным повторением
- Автоматическая генерация из текста документа или фотографии конспекта (мультимодальная модель)
- Алгоритм SM-2 (SuperMemo 2) для оптимального планирования повторений
- Когнитивный анализ: система определяет ваше лучшее время для учёбы, сильные и слабые темы

### 🌌 Созвездие знаний
Интерактивная 3D-визуализация вашей базы знаний:
- Каждый документ — звезда с процедурной текстурой (3 слоя: внешнее свечение → цветное гало → горячее ядро)
- 5 уникальных цветовых палитр: gold, cyan, lavender, rose, mint
- Связи между документами рассчитаны на основе семантической близости средних эмбеддингов
- Адаптивная физика графа: компактные кластеры для малого числа документов, масштабирование для больших
- Процедурная туманность на GLSL-шейдерах, звёздное поле из 5000 частиц, bloom-эффект
- Клик по связи показывает общие термины и самые похожие фрагменты между документами

### ⚡ Индексация и поиск
- **Локальные эмбеддинги**: EmbeddingGemma 300M (ONNX/WASM) — полностью на устройстве
- **Батчевая обработка**: индексация по 4 чанка за раз для GPU-эффективности
- **Фоновая индексация**: не блокирует UI, с live-прогрессом `⚙️ [2/5] document.pdf — 64% (16/25)`
- **Предобработка текста**: очистка от URL, hex-кодов, технического мусора для чистых эмбеддингов
- **Извлечение ключевых слов**: TF-IDF-подобный алгоритм с фильтрацией стоп-слов и шума

### 🔮 Serendipity Engine — «Инсайт дня»
Каждый день Lumea находит неожиданную, но реальную связь между вашими документами из разных дисциплин. Алгоритм:
1. Случайно сэмплирует пары фрагментов из разных документов
2. Ищет «золотой диапазон» схожести (0.25–0.55) — достаточно похожи, чтобы быть связанными, но достаточно разные, чтобы быть _интересными_
3. Просит LLM сформулировать элегантную междисциплинарную связь

### 🧬 Умная память
Система в фоне анализирует взаимодействия пользователя и извлекает долгосрочные факты: стиль обучения, пробелы в знаниях, интересы. Эти факты используются для персонализации RAG-ответов.

## Быстрый старт

```bash
# Установить зависимости
npm install

# Настроить API-ключ
cp .env.example .env
# Вписать VITE_GEMINI_API_KEY в .env

# Запустить
npm run dev
```

> **Примечание:** Семантический поиск, индексация и OCR работают без API-ключа — модель эмбеддингов и Tesseract загружаются локально. API-ключ нужен только для генеративных функций (конспекты, RAG-ответы, флеш-карточки, Serendipity Engine).
